# Variáveis - JavaScript | Senai

Este repositório contém exemplos e exercícios sobre **variáveis em JavaScript**, desenvolvidos durante os estudos no **Senai**.

##  Estrutura do Projeto

- `index.js` → Arquivo principal para execução dos exemplos.
- `declaracao.js` → Exemplos de declaração e inicialização de variáveis.
- `escopo.js` → Exemplos de escopo de variáveis (local e global).
- `.gitignore` → Arquivos ignorados pelo Git.

##  Objetivo

O objetivo deste projeto é praticar conceitos básicos de **JavaScript**, como:

- Declaração de variáveis
- Inicialização de variáveis
- Escopo de variáveis
- Organização de código

##  Como executar

1. Instale o **Node.js** no seu computador.
2. Abra o terminal na pasta do projeto.
3. Execute o arquivo desejado:
