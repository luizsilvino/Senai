#  Estudos de Lógica em JavaScript

Este repositório foi criado com o objetivo de praticar **lógica de programação em JavaScript**, com foco em estruturas fundamentais como:

* 🔁Laços de repetição (`for`, `while`)
*  Vetores (arrays)
*  Matrizes
*  Clean Code
*  Resolução de problemas

---

##  Estrutura do Projeto

```
 estudos-js
├── CleanCode/
├── for/
├── while/
├── vetores/
├── matrizes/
├── desafios/
├── index.html
├── script.js
└── .gitignore
```

##  Conteúdo

###  Laços de Repetição

Exercícios utilizando:

* `for`
* `while`

Objetivo: aprender repetição, contagem, loops e controle de fluxo.

---

###  Vetores (Arrays)

* Manipulação de listas
* Filtragem de dados
* Uso de `.map()`, `.filter()` e `.reduce()`

---

###  Matrizes

* Arrays dentro de arrays
* Percorrer linhas e colunas
* Resolver problemas em grade (ex: jogos)

---

###  Clean Code

* Escrita de código limpo e organizado
* Boas práticas de nomenclatura
* Funções pequenas e reutilizáveis

---

###  Desafios

Problemas práticos para aplicar tudo o que foi aprendido.

## ▶ Como usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/estudos-js.git
```

2. Abra o projeto:

* Pode abrir o `index.html` no navegador
* Ou usar o VS Code

3. Execute os arquivos `.js` pelo navegador ou Node.js
